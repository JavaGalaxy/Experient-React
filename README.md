# User Autocomplete App

A React application with Material UI autocomplete that fetches user data from JSONPlaceholder API and displays formatted names and addresses.

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5174`

## Features

- Search users by name with autocomplete
- Name formatting: "Last, First (Title)" format
- Display user address when selected
- Built with React + TypeScript + Vite

## Development

- Lint code: `npm run lint`
- Build project: `npm run build`
